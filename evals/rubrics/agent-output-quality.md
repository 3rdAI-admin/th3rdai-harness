# Rubric: Agent Output Quality

Score each category from 1 to 5.

## Criteria

- Directness
- Correctness
- Completeness
- Evidence and traceability
- Format adherence
- Actionability
- Appropriate uncertainty
- Handoff readiness

## Pass Threshold

An output passes when:

- It answers the request directly.
- Claims are supported by project context or clearly marked as assumptions.
- The next action is obvious.
- Required output format is followed.

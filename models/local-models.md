# Local Models

Use local models when privacy, offline work, or cost control is more important than peak model quality.

## Candidate Uses

- Drafting initial plans
- Summarizing local documentation
- Running low-risk review passes
- Comparing outputs against hosted models

## Common Local Providers

- Ollama
- LM Studio
- llama.cpp-based runners

## Evaluation Requirement

Do not assume a local model is suitable for a workflow until it has been tested against representative eval cases.
